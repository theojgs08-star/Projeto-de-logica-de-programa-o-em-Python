Estou estudando lógica de programação na faculdade e desenvolvendo minhas habilidades principalmente com Python, que tem sido a linguagem base para praticar e consolidar os conceitos aprendidos em sala de aula. O foco do meu aprendizado não é apenas escrever código, mas entender como pensar de forma estruturada para resolver problemas de maneira lógica, organizada e eficiente.

Durante as aulas e atividades práticas, estou aprendendo conceitos fundamentais como variáveis, tipos de dados, operadores, estruturas condicionais e estruturas de repetição. Esses elementos são essenciais para construir qualquer programa, pois permitem controlar o fluxo da execução e criar soluções que se adaptam a diferentes situações. Além disso, estou aprofundando o uso de funções, que ajudam a organizar melhor o código, evitar repetições e tornar os programas mais modulares e fáceis de manter.

Outro ponto importante do meu aprendizado é o desenvolvimento de projetos práticos em Python, onde aplico a lógica de programação em situações mais próximas do mundo real. Nesses projetos, trabalho com sistemas como controle de estoque, bancos digitais, bibliotecas e outros sistemas de gestão. Isso me ajuda a entender como a lógica se transforma em aplicações reais utilizadas no mercado de tecnologia.

Também estou aprendendo sobre estruturas de dados como listas, dicionários e conjuntos, que são fundamentais para organizar informações dentro dos programas. Esses recursos permitem armazenar e manipular dados de forma eficiente, algo essencial para qualquer sistema moderno. Junto com isso, estou praticando a manipulação de arquivos, como JSON, para simular bancos de dados e entender como os sistemas armazenam informações de forma persistente.

Além da parte técnica, estou desenvolvendo meu raciocínio lógico e minha capacidade de resolver problemas. A lógica de programação me ensina a dividir problemas complexos em partes menores, analisar cada etapa e construir soluções passo a passo. Isso é uma habilidade importante não só para programação, mas também para outras áreas da tecnologia e da vida profissional.

Outro aspecto relevante é a introdução à programação orientada a objetos (POO), onde começo a entender como representar entidades do mundo real dentro do código, utilizando classes e objetos. Isso torna os sistemas mais organizados, escaláveis e próximos da realidade de aplicações profissionais.

De forma geral, meu aprendizado em lógica de programação com Python está sendo fundamental para construir uma base sólida na área de tecnologia. Estou conseguindo unir teoria e prática, desenvolvendo projetos cada vez mais completos e entendendo como funciona a estrutura por trás dos sistemas que usamos no dia a dia, como aplicativos, sites e sistemas empresariais.
